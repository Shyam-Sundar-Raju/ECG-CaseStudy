# Contributing to ECG & Heart Rate Live Visualization (Python GUI)

Thank you for your interest in contributing to this project! Your contributions help improve the project, fix issues, and add new features.  

## How to Contribute

1. **Fork the repository** to your GitHub account.
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/your-username/repo-name.git

3. **Create a new branch** for your feature or bugfix:

   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes**. Ensure your code is clean, well-commented, and follows Python best practices.
5. **Test your changes** thoroughly before committing.
6. **Commit your changes** with a clear message:

   ```bash
   git commit -m "Add feature: description"
   ```
7. **Push your branch**:

   ```bash
   git push origin feature/your-feature-name
   ```
8. **Open a Pull Request** and describe your changes clearly.

---

## Code Guidelines

* Follow [PEP8](https://www.python.org/dev/peps/pep-0008/) for Python coding style.
* Use descriptive variable and function names.
* Keep functions modular and concise.
* Update or add comments/documentation where needed.

---

## Issues and Bugs

* Before creating a new issue, check existing issues to avoid duplicates.
* Provide a detailed description of the bug or feature request.
* Include screenshots if relevant (especially for GUI issues).

---

## Feature Requests

* Feel free to propose new features or improvements.
* Discuss your idea in an issue before implementing it.

---

## Getting Help

For any questions about the project setup or running the code:

* Ensure Python 3.8+ and required dependencies (`pyserial`, `matplotlib`, `numpy`) are installed.
* Verify the STM32 + AD8232 + HC-05 setup and serial port configuration.
* Refer to the [README](README.md) for detailed instructions.
